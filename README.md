# EcoPulse

Carbon-aware data center scheduling system that reduces emissions by shifting flexible workloads to low-carbon electricity periods.

## Project Overview

EcoPulse uses grid emissions forecasting to make intelligent scheduling decisions for data center workloads. By predicting carbon intensity and identifying low-emission windows, the system helps reduce avoidable carbon emissions without requiring new infrastructure.

## Key Features

- Short-horizon carbon intensity forecasting (1-24 hours)
- Policy-driven scheduling for flexible workloads
- Real-time grid and weather data integration
- Emissions impact reporting and analysis
