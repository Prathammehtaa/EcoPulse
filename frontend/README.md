# EcoPulse React Frontend

This is a separate React version of the current EcoPulse UI.

## Quick Run

1. Open a terminal in `frontend-react`
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Open the URL shown in the terminal, usually `http://localhost:5173`

## Notes

- This does not modify the existing Streamlit UI.
- It mirrors the current login, sidebar, dashboard, scheduler, impact, alerts, and admin pages using React.
- It currently uses mock data, matching the current frontend state.
